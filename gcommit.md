## git commit

**git commit**

```bash=
     git commit 


```
это команда для записи индексированных изменений в репозиторий Git.

# Создать новый коммит 

```bash=
     git commit  -m'сообщение'


```
# Включить новые изменения в последний созданный коммит

```bash=
    git commit --amend [-m'сообщение']



```
